# Dokumentasi QuickTix

Folder ini berisi dokumentasi lengkap untuk sistem QuickTix.

## Daftar Dokumentasi

### 1. QR_VALIDATION_GUIDE.md
Panduan lengkap tentang cara kerja validasi QR tiket di sistem QuickTix.

**Isi:**
- Cara kerja validasi QR
- Perbedaan halaman admin vs user
- Troubleshooting masalah umum
- Konfigurasi keamanan
- Contoh penggunaan

### 2. DEPLOYMENT.md (di root folder)
Panduan deployment dan instalasi sistem QuickTix.

### 3. QUICK_START.md (di root folder)
Panduan cepat untuk memulai menggunakan QuickTix.

### 4. README.md (di root folder)
Dokumentasi utama sistem QuickTix.

## Cara Menggunakan Dokumentasi

1. **Untuk Developer**: Baca semua dokumentasi untuk memahami sistem secara menyeluruh
2. **Untuk Admin**: Fokus pada QR_VALIDATION_GUIDE.md dan DEPLOYMENT.md
3. **Untuk User**: Baca QUICK_START.md untuk panduan penggunaan
